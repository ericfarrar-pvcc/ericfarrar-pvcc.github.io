Chapter 4 Terms:
TERM DEFINITION
Network Classifications
  • LAN: local area netowrk
  • MAN: metropolitan area network
  • WAN: wide area network
Network Models
  • Bus Network: Machines are all connected to a common communication
  • Star Network: One Machine serves as a central focal point to which all others are connected
Protocols
  • CSMA/CD Ethernet: This protocol dictates that each message be broadcast to all the machines on the bus
  • CSMA/CA Wifi: wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them.
Combining Networks
  • repeater: a device that passes signals back and forth between the two original buses without considering the meaning of the signals
  • bridge: Connects two buses, does not necessarily pass all messages across the connection
  • switch: a bridge with multiple connections
  • router: special purpose computers used for forwarding messages
  • gateway: The point at which one network is linked to an internet
Client/Server Model
  • p2p: peer-to-peer: client/server model involves one process (the server) providing a service to numerous others (clients)
  • Distributed system: consist of software units that execute as processes on different computers.
  • Cloud Computing: huge pools of shared computers on the network can be allocated for use by clients as needed
Internet Addressing
  • Tier 1: very high-speed, high-capacity, international WANs
  • Tier 2: more regional in scope and less potent in their capabilities
  • Tier 3: independent Internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.

Internet is:
  • packet-switching network: digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data.
  • fault tolerant: Will not fail entirely if components fail
  • redundant: Multiple pathways for data in event of a failure
Cybersecurity
  • malware: Malicious Software
  • virus: software that infects a computer by inserting itself into programs that already reside in the machine.
  • worm: autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • Trojan horse: a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.
  • Spyware: software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • Phishing: a technique of obtaining information explicitly by simply asking for it.
  • DDoS attack: the process of overloading a computer with messages.
Protection
  • Firewall: A primary prevention technique to filter traffic passing through a point in the network
  • Spam filter: firewalls designed to block unwanted email.
  • Proxy server: a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • Antivirus software: used to detect and remove the presence of known viruses and other infections.
  • Crytography: the study of sending and receiving messages securely in the presence of adversaries.
  • Public Key Encryption: involves techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
