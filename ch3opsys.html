Operating Systems Terminology
TERM DEFINITION
Operating System An operating system is the software that controls the overall operation of a computer
Job Queue a storage organization in which objects (in this case, jobs) are ordered in first-in, first-out fashion
FIFO First in, First out
Interactive Processing operating systems that allowed a program being executed to carry on a dialogue with the user through remote terminals
Time Sharing operating systems that provide service to multiple users at the same time. time-sharing refers to multiple users sharing access to a common computer, whereas multitasking refers to one user executing numerous tasks simultaneously
Multiprogramming time is divided into intervals and then the execution of each job is restricted to only one interval at a time
Application Software consists of the programs for performing tasks particular to the machines utilization
System Software performs those tasks that are common to computer systems in general
Utility Software programs for performing activities that are fundamental to computer installations but not included in the operating system
Graphical User Interface objects to be manipulated, such as files and programs, are represented pictorially on the display as icons
Shells communicated with users through textual messages using a keyboard and monitor screen
Kernel the internal part of an operating system
File manager coordinates the use of the machines mass storage facilities
Device Drive the software units that communicate with the controllers
Memory Manager charged with the task of coordinating the machine’s use of main memory
Virtual Memory large “fictional” memory space created by paging
Boot-Strapping how the operating system gets started
Read-only-memory (ROM) a small portion of a computer’s main memory where the CPU expects to find its initial program is constructed from special nonvolatile memory cells
Dead-Lock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling holding data for output at a later but more convenient time
Super user (admin) This person gains highly privileged access to the operating system
Multi-Factor Authentication the user must provide a password, and also a special access code text message to the user’s smartphone during the login process
